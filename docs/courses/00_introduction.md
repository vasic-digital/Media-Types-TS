## Module Overview

This module provides the shared TypeScript type definitions used throughout the Catalogizer ecosystem. Understanding these types is the foundation for working with any Catalogizer API or UI component.

## What You Will Learn

1. The three domains: auth, media, and collections
2. How `MediaEntity` represents the hierarchical media model
3. Working with `PaginatedResponse<T>` generics
4. Smart collection rules and their operators
5. Why pure type packages have zero runtime cost

## Prerequisites

- TypeScript 5.0+
- Basic understanding of TypeScript interfaces

## Estimated Lessons

5 lessons, ~30 minutes total
